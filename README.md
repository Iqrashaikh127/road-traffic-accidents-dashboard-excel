# Road Traffic Accidents Dashboard (Advanced Excel)

## Project Overview

The **Road Traffic Accidents Dashboard** is an interactive dashboard developed using **Microsoft Excel** to analyze road accident data. The dashboard provides insights into accident trends, casualty statistics, vehicle categories, and road conditions through interactive visualizations.

## Objectives

- Analyze road accident trends.
- Monitor casualty statistics.
- Identify accident patterns based on vehicle type and road conditions.
- Provide interactive data exploration using Excel dashboards.

## Technologies Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Validation

## Dashboard Features

- Total Accidents Overview
- Casualty Analysis
- Vehicle Type Analysis
- Monthly Accident Trends
- Road Surface Analysis
- Interactive Filters using Slicers
